# Outlier - Autism & Disability Learning Platform

Educational platform for children with autism and disabilities featuring AI-powered learning, gamification, and comprehensive support resources.

## Features

- **Education Center**: Flash cards, videos, courses, and AI assistant
- **Funding Resources**: Access to funding programs and financial assistance
- **Shop**: Curated autism-specific products
- **Progress Reports**: Data summarization for parents and organizations
- **AI-Based Learning**: Speech recognition and adaptive content
- **Gamified Learning**: Engaging interactive activities

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons
